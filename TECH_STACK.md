# 技術選定

## フロントエンド

- **Next.js 16** - React フレームワーク（App Router）
- **React 19** - UI フレームワーク
- **Tailwind CSS 4** - ユーティリティファーストCSS
- **TypeScript 5** - 静的型付け言語

## バックエンド

- **NestJS 11** - Node.js フレームワーク
- **GraphQL (schema-first)** - API スキーマ駆動開発
- **Apollo Server 5** - GraphQL サーバー
- **PostgreSQL** - リレーショナルデータベース
- **Prisma 7** - ORM・マイグレーションツール

## 開発ツール

- **pnpm 10** - 高速パッケージマネージャー
- **Biome** - リント・フォーマッター（ESLint+Prettier の代替）
- **Jest** - テストフレームワーク
- **Docker / Docker Compose** - コンテナ化

## 実行環境

- **Node.js 22 LTS** - JavaScript ランタイム
